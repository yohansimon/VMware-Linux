# VMware-Linux
https://github.com/yohansimon/VMware-Linux/issues/1#issue-3670850634
Installation de VMware sous Debian, Ubuntu...
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install build-essential gcc-12 libgcc-12-dev -y
```
Vous Télécharger le logiciel VMware pour linux sur leur site puis vous lancez ces deux commandes suivantes.
```bash
sudo chmod +x VMware-Workstation-Full-17.6.4-24832109.x86_64.bundle
sudo ./VMware-Workstation-Full-17.6.4-24832109.x86_64.bundle
```
